Software prerequirements:
* Visual Studio 2015
* Windows Azure SDK for .NET 2.7+
* Ruby 1.9.3+
* Node.js 0.12+
* GitBash for Windows

Post checkout steps:
* gem install compass
* npm install -g grunt
* npm install -g bower

Post update steps:
* npm install
* npm run-script bower
* grunt post-checkout
